# Rydr IOS 🚗
- Rydr is an IOS app that helps first time drivers with logging their 50 hours for their driver's license.
- Users are able to start a timer within the app that runs while the device is offline and saves to a collection to show their time spent driving.
- Users can also manually add time via a button to log hours they've done before using Rydr to incase they forgot to start the timer.
- Users can also view a tab that shows driving tips with videos to aid them in becoming amazing drivers!
